🖥️ الاتصال بمنصة MT5

أدخل بيانات خادم VNC للوصول إلى منصة MetaTrader 5

ℹ️

يجب أن يكون لديك خادم Windows VPS مع MT5 مثبت عليه و VNC Server نشط. أدخل عنوان الخادم أدناه للاتصال.

📋 خطوات إعداد الخادم

  • احصل على Windows VPS (مثل Contabo, Hetzner, Vultr)
  • ثبّت MetaTrader 5 على الخادم
  • ثبّت TightVNC Server أو TigerVNC
  • ثبّت noVNC و websockify للوصول عبر الويب
  • شغّل websockify: websockify --web /usr/share/novnc 6080 localhost:5900

🤖 إضافة روبوت تداول (Expert Advisor)

  • بعد الاتصال بالمنصة، افتح MT5
  • انتقل إلى: File → Open Data Folder → MQL5 → Experts
  • انسخ ملف الروبوت (.ex5 أو .mq5) إلى مجلد Experts
  • أعد تشغيل MT5 أو اضغط Refresh في Navigator
  • اسحب الروبوت على الرسم البياني المطلوب

🔒 نصائح الأمان

  • استخدم HTTPS دائماً للاتصال
  • قم بتعيين كلمة مرور قوية لـ VNC
  • استخدم VPN إضافي إن أمكن
  • فعّل جدار الحماية وافتح المنافذ المطلوبة فقط
